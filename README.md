# AI Chatbot (Python)

This project is a simple AI chatbot built using Python. It can understand user input and respond based on predefined logic and basic NLP techniques.

---

## Tech Stack
- Python
- NLP (basic text processing)
- Regex / String matching

---

## Features
- Handles user queries with predefined responses  
- Simple natural language understanding  
- Interactive command-line chatbot  
- Easy to extend with more responses  

---

## How It Works
1. User enters a message  
2. Chatbot processes input using text matching  
3. Matches input with predefined patterns  
4. Returns appropriate response
